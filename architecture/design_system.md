# Design System SOP: Bio-Digital Minimalism

## 1. Core Philosophy
*   **Aesthetic**: "Bio-Digital Minimalism"
*   **Tone**: Clean, Trustworthy, Efficient, Medical-Grade.
*   **Target**: High-contrast for low-light rural settings; large touch targets for mobile.

## 2. Color Palette (Derived from References)
### Primary
*   **Deep Medical Blue**: `#122056` (Headers, Primary Actions)
*   **Pulse Accent**: `#5B65DC` (Highlights, Active States)
*   **Success Green**: `#10B981` (Vitals Normal, Success)
*   **Alert Red**: `#EF4444` (Critical Vitals, Errors)

### Neutrals
*   **Background**: `#F8FAFC` (Slate 50)
*   **Surface**: `#FFFFFF` (White Cards)
*   **Text Main**: `#1E293B` (Slate 800)
*   **Text Muted**: `#64748B` (Slate 500)

## 3. Typography
*   **Font Family**: `Poppins` (Sans-serif).
*   **Scaling**:
    *   **H1**: 24px (Mobile) / 32px (Desktop) - Bold
    *   **H2**: 20px / 24px - SemiBold
    *   **Body**: 16px - Regular
    *   **Caption**: 14px - Medium

## 4. UI Components
### Cards
*   White background, `rounded-xl`, soft shadow (`shadow-sm`).
*   Padding: `p-4` (Mobile), `p-6` (Desktop).

### Buttons
*   **Primary**: Deep Medical Blue background, White text, `rounded-lg`.
*   **Secondary**: Transparent background, Blue text, Border `border-slate-200`.

### Inputs
*   Background `bg-slate-50`, Border `border-slate-200`.
*   Focus ring: `ring-2 ring-blue-500`.

## 5. Mobile Responsiveness
*   **Bottom Navigation**: Fixed for Patient Portal.
*   **Touch Targets**: Minimum 44px height for all interactive elements.
